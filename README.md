## Overview
Travel to a brand-new dimension.

Mine a new ore that can be transformed into a portal, allowing you to travel to a mysterious dimension called “Novarite.”


## Features
- Tools
- Armor
- Trees
- Blocks
- Food
- Creatures
- Structures
